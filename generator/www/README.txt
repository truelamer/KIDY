Планы генерации содержатся в папке plans в формате php. Название файла есть название плана.

Какие планы там есть:
clear_data.php - очистка базы данныз
zero_profit.php - план генерации данных, аналитический запрос к которым показывает нулевую прибыл (7 задание)

Запуска генератора через файл index.php
